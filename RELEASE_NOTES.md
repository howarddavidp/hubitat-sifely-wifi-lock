# Sifely WiFi Lock Driver v1.0.0

This is the first production release of the unofficial Sifely S WiFi Lock cloud driver for Hubitat. It is promoted from the successfully tested internal build **0.3-test Version D**.

## Confirmed working

- Manual lock and unlock
- Refresh and physical-state synchronization
- Four-hour proactive token refresh
- Fresh authentication before lock/unlock when enabled
- Authentication failure detection and retry
- List Locks, including names, Lock IDs, and online status
- Multiple locks
- Hubitat Basic Rules
- Geofence arrival unlocking
- Geofence departure locking

## Planned

A clearer selectable authentication mode is planned for version 1.5.0.

This is an unofficial community driver and is not affiliated with Sifely or Hubitat.
